Simulação Campeonato Brasileiro

Introdução

Esse projeto tem como função, simular um campeonato brasileiro.
A simulação segue os moldes do campeonato oficial, ou seja, segue o padrão de acordo com a CBF.

Menu da Simulação

  • Cadastrar Campeonato
  • Limpar Campeonato
  • Listar Campeonato
  • Cadastrar Time
  • Listar Time
  • Limpar Time
  • Sortear Times
  • Sortear Resultados
  • Listar Partidas
  • Ver Resultados
  • Ver Vencedor
  • Regras CBF
